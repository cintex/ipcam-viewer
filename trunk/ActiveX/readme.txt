How to create ActiveX control from VCL Component
================================================

Create
------
Register VCL component
Create ActiveX Project
Link .pas source code
Launch wizard to create an ActiveX control and fill all required fields (see readme.png)

Update
------
Remove auto-created files from the project like:

* IPCamViewer.ridl
* IPCamViewer.tlb
* IPCamViewer.tlb
* IPCamViewer_TLB.pas
* IPCamViewerAx.pas

And repeat "Create" step
